# Screenshots

Drop three images in this folder to complete the main README's Screenshots section:

- `original.png` — an unsolved puzzle screenshot as uploaded
- `detected-grid.png` — the same screenshot with the detected grid/clues visible (optional;
  can be the same as `original.png` if you don't have an annotated version)
- `solved.png` — the app's rendered result after solving

Once added, the table in the root [README.md](../../README.md#screenshots) will display them.
